# MFA Business Intelligence - Prototipo Ejecutivo v2.7

## Objetivo
Versión que combina:
- Diseño ejecutivo del prototipo v2.3/v2.4.
- Carga directa de Excel de v2.6.
- Motor por último nivel disponible para evitar duplicar cuenta mayor + subcuenta + auxiliar.

## Cómo usar
1. Abre el módulo **Carga de Datos**.
2. Carga tu archivo `.xlsx` o `.xlsm` con hojas:
   - BALANZA
   - DICCIONARIO MFA
   - BANCOS
3. El dashboard se actualiza en navegador.

## Nota
La carga de Excel ocurre en el navegador. No guarda información en servidor.
