# MFA Business Intelligence - Prototipo Ejecutivo v2.4

## Estructura
- `index.html`: interfaz principal.
- `css/main.css`: identidad visual y estilos.
- `js/app.js`: navegación, filtros y datos demo.
- `assets/`: logo y favicon.
- `data/`: carpeta preparada para futuros archivos JSON o Excel procesados.

## Cómo subir a Vercel
1. Entra a https://vercel.com
2. Crea una cuenta o inicia sesión.
3. Selecciona **Add New Project**.
4. Sube esta carpeta completa.
5. Vercel detectará un sitio estático.
6. Da clic en **Deploy**.
7. Comparte la URL pública.

## Nota
Esta versión es un prototipo ejecutivo visual con datos demo anonimizados. La conexión real con balanza, bancos, clientes y proveedores se desarrollará en la siguiente fase.
